# Mumbai LULC Report Generation

This repository contains the analysis notebook and report for **LULC Change Detection and Urban Heat Island Analysis of Mumbai BMC Area (2015–2025) with CA-Markov 2030 & 2035 Prediction**.

## Files

| File | Description |
|------|-------------|
| `Mumbai_LULC_Final_v5 (10).ipynb` | Google Colab notebook — complete analysis code |
| `Mumbai_LULC_Report.md` | Full report in Markdown — follows the exact format of the reference PDF |
| `Mumbai_LULC_Report_Final.pdf` | Reference report PDF (template format) |
| `generate_report.py` | Script to auto-populate report with values from notebook outputs |

## How to Generate the Complete Report

### Step 1: Run the Colab Notebook

1. Open the notebook in Google Colab:
   [Mumbai LULC Analysis](https://colab.research.google.com/drive/1lMBuZtxwnAauOBXYu-L_7DdSyNA_utd1?usp=sharing)

2. Run all cells sequentially (**Runtime → Run all**).
   - You will need to authenticate with Google Earth Engine.
   - Full execution takes approximately 15–20 minutes.

3. Download the notebook **with outputs preserved**:
   **File → Download → Download .ipynb**

4. Save the output images generated by the notebook:
   - `accuracy_assessment.png`
   - `lulc_maps.png`
   - `lst_maps.png`
   - `markov_transition.png`
   - `lulc_predicted_maps.png`
   - `lulc_area_trend.png`
   - `lst_per_class.png`
   - `ndvi_vs_lst.png`
   - `ndbi_vs_lst.png`
   - `ndwi_vs_lst.png`

### Step 2: Auto-Populate the Report (Optional)

```bash
python generate_report.py "Mumbai_LULC_Final_v5 (10).ipynb"
```

This reads the notebook outputs and replaces `*[notebook]*` placeholders in `Mumbai_LULC_Report.md` with actual values, producing `Mumbai_LULC_Report_Populated.md`.

### Step 3: Insert Figures

Replace the `*(Insert ... from Colab output)*` markers in the report with the actual `.png` images from Step 1.

### Step 4: Convert to PDF (Optional)

Use any Markdown-to-PDF converter:

```bash
# Using pandoc
pandoc Mumbai_LULC_Report.md -o Mumbai_LULC_Report.pdf

# Using grip (GitHub-flavoured Markdown)
pip install grip
grip Mumbai_LULC_Report.md --export Mumbai_LULC_Report.html
```

## Report Structure

The report follows the exact format of the reference PDF (`Mumbai_LULC_Report_Final.pdf`):

1. **Title Page** — project details and submission info
2. **Table of Contents** — with page references
3. **List of Figures and Tables**
4. **Abstract** — complete summary of methodology and findings
5. **Chapter 1: Introduction** — background, problem statement, objectives, scope
6. **Chapter 2: Literature Review** — LULC studies, RF classification, UHI indices
7. **Chapter 3: Study Area** — location, physical characteristics, climate
8. **Chapter 4: Data and Methodology** — data sources, tools, flowchart, classification system, RF parameters, CA-Markov method
9. **Chapter 5: Results and Discussion** — LULC maps, accuracy, area statistics, LST analysis, correlation analysis, predictions
10. **Chapter 6: Conclusion** — findings, recommendations, limitations, future scope
11. **References** — cited literature
12. **Appendix** — notebook reference, abbreviations, output file list

## About LIME Images

The report includes a dedicated section (§4.11 and §5.10) for LIME (Local Interpretable Model-agnostic Explanations) results. If your Colab notebook includes LIME analysis:

1. Save the LIME explanation images from the notebook
2. Insert them at the marked locations in Chapter 5
3. Fill in the LIME methodology section in Chapter 4

LIME images typically include:
- Feature importance bar charts for individual pixel predictions
- Superpixel explanation maps showing which image regions support/contradict each classification
- Comparison across different LULC classes
